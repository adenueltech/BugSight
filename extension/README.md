## Usage

### Explain Selected Error

1. Select an error message in your editor
2. Open Command Palette (Ctrl+Shift+P)
3. Run: `BugSight: Explain Error`
4. View the AI explanation in a side panel

### View Error History

- Click the BugSight icon in the Activity Bar
- Or run: `BugSight: Show Error History`

### Clear History

- Run: `BugSight: Clear History`

## Commands

- `BugSight: Explain Error` - Explain selected error text
- `BugSight: Show Error History` - View all analyzed errors
- `BugSight: Clear History` - Clear error history

## Requirements

- VSCode 1.85.0 or higher

## Privacy

- Error data is processed securely by BugSight's AI service
- No personal data is stored
- History is stored locally in VSCode
